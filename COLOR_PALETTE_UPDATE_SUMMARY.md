## Design Philosophy

The new color scheme creates a **sophisticated, luxurious, and classy** aesthetic:

### Primary Design Elements

1. **Backgrounds**: Deep, rich gradients transitioning from noir to teal create depth and elegance
2. **Accents**: Burgundy provides bold, refined contrast  
3. **Highlights**: Gold adds warmth and luxury, drawing attention to CTAs and important elements
4. **Text**: Pearl (light gray) ensures excellent readability against dark backgrounds

### Visual Effects Applied

- **Gradients**: Smooth transitions between burgundy, gold, and burgundy for visual interest
- **Glows**: Gold text shadows and glowing effects on headlines
- **Glass Morphism**: Translucent teal backgrounds with subtle gold borders
- **Shadows**: Gold-tinted shadows on cards and buttons for depth
- **Shimmer Animation**: Gradient text with animated shimmer effect

### Button Styles

- **Primary**: Burgundy gradient background with gold hover state and gold shadows
- **Secondary**: Teal glass with gold border that intensifies on hover

---

## User Experience Enhancements

1. **Contrast**: Excellent text readability with pearl on dark backgrounds
2. **Focus States**: Clear gold focus rings on all interactive elements  
3. **Hover Effects**: Smooth transitions to gold highlights
4. **Loading States**: Gold gradient loading animations
5. **Visual Hierarchy**: Gold draws eyes to CTAs while burgundy provides elegant accents

## Technical Implementation

All colors use Tailwind CSS custom color tokens:
- `bg-noir`, `bg-teal`, `bg-burgundy`
- `text-pearl`, `text-gold`  
- `border-gold`, `ring-gold`
- Custom opacity variants (`/80`, `/70`, `/60`, etc.)

Gradients use the `from-`, `via-`, `to-` pattern for smooth multi-color transitions.
